## 📖 Overview
This free, single-file CLI tool performs local n-gram frequency analysis to identify missing high-value phrases in your content compared to competitors. It solves the problem of expensive SaaS subscriptions and complex API integrations by using only standard libraries, ensuring instant results and privacy preservation. Designed for intelligence gathering and strategic alignment, the tool supports Markdown, HTML, and TXT files to highlight gaps without external LLM inference costs.
